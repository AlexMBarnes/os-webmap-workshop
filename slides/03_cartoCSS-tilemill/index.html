What is CartoCSS?
A styling language used in TileMill/Mapbox Studio
It may be helpful to look at HTML/CSS relationship
Use CSS to select HTML and element/tag/class/id and add style
example
CartoCSS is similar in syntax & function
#layer { symbolizer: value; }
Allows you to target map features...
...and manipulate various aspects of their appearance
You can style...
points, lines, & polygons
rasters
text
buildings
and more!
And there are lots of symbolizing options for different types of features
(see Reference)
What is TileMill?
Map design program
Can handle a variety of geographic data formats

Vector

  • shp
  • kml
  • csv
  • geoJSON
  • PostGIS
Raster: GeoTIFF

Exports to a variety of formats

  • png
  • svg
  • pdf
  • mapnik XML
  • mbtiles
NOT a GIS
geoprocessing
spatial analysis
data management
Best suited for design
